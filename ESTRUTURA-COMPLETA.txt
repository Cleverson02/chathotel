📋 COMO USAR
════════════════════════════════════════════════════════════════════════════

1. INICIAR LOCALMENTE

   $ cd C:\Users\Cleverson\projetos\chathotel
   $ npm install
   $ npm run dev

   Acesse: http://localhost:3000


2. TESTAR FORMULÁRIO

   • Role até "FALE COM A GENTE"
   • Preencha com dados
   • Clique "Agendar demonstração"
   • Observe confirmação
   • Verifique emails (seu + cleverson@...)


3. FAZER DEPLOY

   → Siga: DEPLOY-VERCEL.md

   Resumido:
   a) Criar repositório GitHub
   b) Fazer push do projeto
   c) Conectar na Vercel
   d) Configurar SENDGRID_API_KEY
   e) Adicionar CNAME no Cloudflare (chathotel → cname.vercel.com)
   f) Conectar domínio na Vercel
   g) Aguardar propagação (5-15 min)


4. CUSTOMIZAR

   • Mudar cores: Painel de tweaks (canto inferior direito)
   • Mudar texto: Editar *.jsx
   • Mudar imagem: Substituir assets/hotel-background.png
   • Mudar email: Editar api/send-proposal.js (linha 5)
